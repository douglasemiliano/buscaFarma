# buscaFarma
Projeto para as disciplinas de Banco de Dados e Engenharia de Software do Mestrado em TI do IFPB

front:

Para rodar o projeto, basta instalar o angular, rodar *npm install* e depois *ng serve*

Banco:
Os dados estão mockados, não temos integração com o back nem com front ainda, para visualizar instale o json server: *npm install json-server*
dirija ate a pasta assets, pelo terminal e inicie o servidor digitando: *json-server --watch db.json*
